# Project Context

## What This Repository Is

- Repository: `agam-lang/.github`
- Purpose: Organization-level default community health files, reusable CI/CD workflows, and the public organization profile.
- Technology: GitHub Actions YAML + Markdown

## Organization Context

This is the infrastructure backbone of the `agam-lang` organization.
All other repositories inherit defaults from here (issue templates, PR templates, contributing guide).
Reusable workflows defined here are called by every other repository's CI pipeline.
