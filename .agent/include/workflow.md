# Workflow

## Core Operating Rules

- Changes to reusable workflows affect every repository in the organization.
- Always test workflow changes on a single repo before propagating.
- The profile README must stay accurate and reflect the current state of all repos.
- Community health files (CoC, Contributing, Security) should follow industry best practices.

## Documentation Contract

- When adding or modifying reusable workflows, update the README with usage instructions.
- Keep the profile README synchronized with actual repository statuses.
