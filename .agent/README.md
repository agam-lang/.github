# Agam Organization Defaults — Agent Board

This directory is the shared source of truth for agent-facing guidance in the `.github` repository.

## Layout

- `include/`: compact shared context for agents.
- `rules/`: organization-wide guardrails.
- `policy/`: structured operating rules.
- `phases/`: what is active and what is next.

## Organization Role

This repository serves as the organization-level defaults provider:
- `profile/README.md`: the public organization landing page
- `.github/workflows/`: reusable CI/CD workflows consumed by all other repos
- Community health files: CODE_OF_CONDUCT, CONTRIBUTING, SECURITY, templates
- Dependabot and automation configurations
